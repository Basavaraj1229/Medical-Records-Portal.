Doctor-Patient Portal

A comprehensive healthcare management system built with Java, JSP, and MySQL

Complete web-based solution for managing appointments, patient records, and healthcare administration with role-based access for Patients, Doctors, and Administrators.

Quick Start Guide

How to Run This Project
📋 Prerequisites:
  • Java JDK 8+
  • Apache Maven
  • MySQL 5.7+
  • Apache Tomcat 7+
🚀 Quick Commands:
git clone https://github.com/Basavaraj1229/Doctor-Patient-Portal-.git
cd Doctor-Patient-Portal-
mvn clean package
mvn tomcat7:run

Access URLs:
• Home: http://localhost:8080/Doctor-Patient-Portal/
• Admin: http://localhost:8080/Doctor-Patient-Portal/admin_login.jsp
• Doctor: http://localhost:8080/Doctor-Patient-Portal/doctor_login.jsp
